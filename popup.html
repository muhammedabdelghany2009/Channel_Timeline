Full guide to downloading YouTube playlists and channels using yt-dlp and PowerShell.
Step 1 — Install yt-dlp
Open PowerShell as Administrator and run:
winget install yt-dlp
Or download the .exe from github.com/yt-dlp and place it in C:\Windows\System32
Step 2 — Install ffmpeg (required for best quality)
Run in PowerShell:
winget install Gyan.FFmpeg
Restart PowerShell, then verify:
ffmpeg -version
Download — Best Quality (1080p)
yt-dlp -f "bestvideo[height<=1080]+bestaudio/best" --merge-output-format mp4 "PASTE_LINK_HERE"
Download — Full Playlist
yt-dlp -f "bestvideo[height<=1080]+bestaudio/best" --merge-output-format mp4 -o "%(playlist_index)s - %(title)s.%(ext)s" "PASTE_PLAYLIST_LINK"
Download — Full Channel
yt-dlp -f "bestvideo[height<=1080]+bestaudio/best" --merge-output-format mp4 -o "%(uploader)s/%(title)s.%(ext)s" "https://youtube.com/@CHANNEL_NAME"
Download — Audio Only (MP3)
yt-dlp -x --audio-format mp3 --audio-quality 0 "PASTE_LINK_HERE"
Download — From Timestamp to Timestamp
Replace 00:01:00 and 00:05:00 with your start and end times:
yt-dlp --download-sections "*00:01:00-00:05:00" "PASTE_LINK_HERE"
Quality options
4K:
yt-dlp -f "bestvideo[height<=2160]+bestaudio/best" --merge-output-format mp4 "LINK"
720p:
yt-dlp -f "bestvideo[height<=720]+bestaudio/best" --merge-output-format mp4 "LINK"
480p:
yt-dlp -f "bestvideo[height<=480]+bestaudio/best" --merge-output-format mp4 "LINK"
Save to a specific folder
yt-dlp -o "C:\Downloads\%(title)s.%(ext)s" "PASTE_LINK_HERE"
Skip already downloaded files
yt-dlp --download-archive downloaded.txt "PASTE_PLAYLIST_LINK"
